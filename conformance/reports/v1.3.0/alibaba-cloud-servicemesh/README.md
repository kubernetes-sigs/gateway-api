# Alibaba Cloud Service Mesh

Alibaba Cloud Service Mesh (ASM) provides a fully managed service mesh platform that is compatible with the community Istio open source service mesh. It simplifies service governance, including traffic routing and split management between service calls, authentication security for inter-service communication, and mesh observability capabilities, thereby greatly reducing the workload of development and operations.
